# easyeffects presets and autoload

note: the presets directory changes if easyeffects is installed using flatpak or not

```bash
# flatpak
$HOME/.var/app/com.github.wwmm.easyeffects/config/easyeffects

# others
$HOME/.config/easyeffects
```

note: these autoload files have hashes in the device names - I'll bet those hashes are not consistent across installs,
or maybe even across usb ports
